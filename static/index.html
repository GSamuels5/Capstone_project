Routes HTTP methods Description
/ GET Render an index.html
Other routes are protected by middleware. You must login first.

all users routes information

Routes HTTP methods Description
/workers GET fetch all users
/:id GET fetch user by ID
/update/:id PATCH update a user
/register POST To add a new user
/delete/:id DELETE To delete a user
/login POST To login a user

'/update/:id PATCH To update a leave day
